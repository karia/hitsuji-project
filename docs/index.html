Construction

マイクラ建築企画

今年のお誕生日を楽しんでいただくため、あのマイクラお誕生日ワールドが返ってきました！

※推定プレイ時間 1時間程度

📖 取扱説明書（クリックして開く）

  1. 1. ダウンロード

    ワールドデータのダウンロードは以下の Google ドライブのリンク先から行うことができます！

    ダウンロードリンク：6月25日 公開予定

  2. 2. zip ファイルの展開

    ダウンロードした zip ファイルの中に charlotte2026 フォルダが入っています。

  3. 3. マインクラフト環境へインストール

    charlotte2026 フォルダをマインクラフトの saves フォルダ（デフォルトであれば C:\Users\(ユーザー名)\AppData\Roaming\.minecraft\saves のディレクトリにあります）へ移動してインストールできます。

  4. 4. マインクラフトを起動して、推奨設定を確認してワールドをプレイ！

    本ワールドは Minecraft Java Edition バージョン 26.1 にて作成されています。26.2 での動作チェックも行っていますが、万全に遊ぶのであれば 26.1 を推奨します。

    今作の描画距離「12チャンク」を目安に作成されています。サウンドの設定も適宜調整をお願いいたします！
