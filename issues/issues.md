# 今後やるべきこと（ロードマップ）

## デザイン刷新：別人が作ったように／より格好よく宇宙感を

姉妹サイト `tsukaima-project`（ななし学園3年5組）と「同じ人が作った」と気づかれないようにしつつ、格好よく・マルチデバイス対応・より宇宙感のあるデザインへ刷新する。

**前提メモ（差別化の勘所）**: 2サイトは配色こそ「明るい学校」と「暗い宇宙」で大きく違うが、骨格がそっくりで、そこが“同じ人”のサインになっている。具体的には ①中央寄せ1カラムにカードを縦積み ②角丸pill型の sticky ナビ ③`eyebrow`＋見出しのカードヘッダ ④`page-loader` スピナー＋`is-loading` でブラー解除 ⑤同一マークアップの画像拡大モーダル ⑥`↗` 付き `btn-link` ⑦`COMING SOON` プレースホルダ、など。**配色を変えるだけでは足りず、レイアウト構造・ナビ・モーション・タイポを作り替えることが差別化の本丸。** 以下は全部やる必要はなく、効果の高いものから選んで実施する。

### ナビゲーション

- [x] **メニューをハンバーガーアイコンにしまう**（最優先）。現状は常時表示の角丸pillナビ（`tsukaima` と同型）。これをハンバーガー＋ドロワー/オーバーレイ展開に変え、骨格の一致を崩す
- [ ] スクロール方向に応じてヘッダーを出し入れ（下スクロールで隠す／上で出す）
- [x] スクロール位置に応じた現在セクションのハイライト（スクロールスパイ）
- [x] ドロワーのキーボード操作・フォーカストラップ・`aria-expanded` などアクセシビリティ対応

### レイアウト構造の脱・テンプレ化

- [x] 「中央1カラム＋カード縦積み」から脱却し、宇宙空間を進むような縦スクロール体験へ（フルブリードのセクション、セクション間の余白リズムの変化）
- [ ] 一部セクションを非対称グリッド／横スクロール／重なりレイアウトにして単調さを解消
- [x] カードの角丸・枠線一辺倒をやめ、グロウ枠・ホログラム面・切り欠き等で質感を差別化
- [x] `card-head` の `eyebrow + 見出し + 下線グラデ` の定型を別表現に置き換える

### 宇宙感の強化（背景・ビジュアル）

- [x] ネビュラ/オーロラ風グラデーションやノイズで奥行きを表現
